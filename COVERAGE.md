# Coverage

This document describes the data coverage of the Croatian Data Protection MCP server.

## Authority

**AZOP — Agencija za zaštitu osobnih podataka**
Croatian Personal Data Protection Agency
<https://azop.hr/>

## Decision Types

| Type | Croatian | Description |
|------|----------|-------------|
| `kazna` | Kazna | Administrative fine / sanction |
| `upozorenje` | Upozorenje | Warning |
| `rješenje` | Rješenje | Decision / ruling |
| `mišljenje` | Mišljenje | Opinion |

## Data Freshness

- Database is periodically updated from official AZOP publications
- Use `hr_dp_check_data_freshness` to verify current record counts and latest dates
- All records include source URLs where available

## Languages

- Primary language: Croatian (`hr`)
- Some documents available in English (`en`)

## Limitations

- Coverage may be incomplete; not all AZOP publications are indexed
- This is a research tool — verify all references against primary sources
